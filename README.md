# Premium E-Commerce Platform

A fully functional, minimalist, and modern E-Commerce web application built using Django and Bootstrap 5. This platform handles everything from product discovery to core cart flows and order tracking.

---

## Key Features

* **Dynamic Catalog:** Product grid listing with responsive premium cards, instant view details, and smooth hover effects.
* **Search & Filters:** Real-time search bar functionality with sidebar category navigation and combined state filtering.
* **Authentication & Profiles:** Complete User registration, secure login/logout hooks, and conditional access layers.
* **Robust Cart Flow:** Dynamic add-to-cart operations, state increments, decrements, and total amount calculation.
* **Checkout & Orders:** Fast checkout routing with direct inventory stock reduction and structured database logging.
* **Order Tracking Dashboard:** Client-side "My Orders" interface for users to check transaction breakdowns and status indicators (Pending/Shipped/Delivered).
* **Product Reviews & Ratings:** Isolated client feedback architecture containing 1-5 star ratings and reviews without database model clashing.
* **Advanced Admin Control:** Comprehensive inline visibility of transactional items linked via relational mappings inside the Django Admin Panel.

---

## Technology Stack

* **Backend:** Django (Python)
* **Frontend:** HTML5, CSS3, Bootstrap 5 (Plus Jakarta Sans Typography)
* **Database:** SQLite (Development standard)
